# Megha Garg — Portfolio

A personal portfolio showcasing my expertise in **Marketing**, **Sales Strategy**, and **Brand Management**.

## How to View

1. Open `index.html` in any web browser.
2. Navigate through sections: About, Journey, Expertise, Projects.
3. Visit the **Contact** page via the navbar or `contact.html`.

## Adding Your Photo

Place your photo as `photo.jpg` in this folder. It will automatically appear in the hero and about sections.

## Structure

| File | Purpose |
|------|---------|
| `index.html` | Main portfolio page |
| `contact.html` | Contact form & details |
| `style.css` | All styling |
| `script.js` | Animations & interactions |
| `photo.jpg` | Profile photo (add your own) |

## Sections

- **Hero** — Introduction with name, tagline, and key stats
- **About** — Background and what drives my approach
- **Journey** — Key milestones and growth path
- **Expertise** — Skills in digital marketing, market research, sales, brand management, analytics, and leadership
- **Projects** — Featured work including brand launches, campaigns, consumer research, and sales optimization
- **Contact** — Reach out via form, email, phone, or social links

## Built With

HTML, CSS, JavaScript — no frameworks, no dependencies, fully responsive.
